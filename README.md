# AI Fraud Detection Dashboard

A premium React + Tailwind + Framer Motion UI for real-time scam analysis.

## Stack
- React + Vite
- Tailwind CSS
- Framer Motion

## Run locally
```bash
npm install
npm run dev
```

## Environment
Create a `.env` file from `.env.example`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

The app calls:
- `POST /analyze`
