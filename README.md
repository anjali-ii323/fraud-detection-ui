# AI Fraud Detection Dashboard

Premium React + Tailwind dashboard UI for real-time fraud analysis.

## Stack
- React + Vite
- Tailwind CSS

## Run locally
```bash
npm install
npm run dev
```

## Environment
Create a `.env` file from `.env.example`:

```bash
VITE_API_BASE_URL=http://localhost:8000
```

The app sends analysis requests to:
- `POST /analyze`
